# Análise SAT Score das Escolas de Nova York

Projeto criado para conclusão de uma das etapas do curso de Python Developer da [Datacamp](https://www.datacamp.com/).

## Objetivo
O objetivo principal deste projeto é demonstrar o tratamento de dados utilizando Python 3, unicamente com pandas. Exploramos o processo de manipulação e análise de dados, buscando o resultado de quais escolas se sairam melhores na disciplina de matemática, score geral somados das três disciplinas (matemática, leitura e escrita) e também seu desvio padr~so e outros detalhes sobre.

## Ferramentas Utilizadas
- Python
- Pandas

## Como Utilizar
1. Clone este repositório.
2. Instale a biblioteca necessária utilizando `pip3 install pandas`.
3. Execute o código para visualizar a análise ou explore o notebook Jupyter fornecido para detalhes adicionais.

## Arquivos do Projeto.
- schools.csv (contém as informações analisadas)
- notebook.ipynb (código usado na análise)
